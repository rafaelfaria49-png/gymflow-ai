# reference/ — referências humanas de rosto (garimpo)

Fotos de **rosto humano real** que servem de **âncora visual** ao avaliar os candidatos. Não são geradas aqui; são garimpadas (Pinterest, Unsplash, editorial, etc.) conforme `docs/avatar-design/KAI_MOODBOARD_v1.md §3 (Face)` e `§4 (Pele)`.

## O que procurar (alvo)
- Retrato masculino real, ~30 anos, **neutro-focado**, confiança serena (`MOODBOARD §3`).
- Mandíbula definida natural, olhos vivos ("tem alguém aí"), barba curta ou bem barbeado, corte moderno.
- Pele que **respira**: poros + SSS + micro-imperfeição + variação de cor (`MOODBOARD §4`).
- Luz de estúdio escura, contraluz, fundo premium (`MOODBOARD §6`).

## O que NÃO entra (descarta)
- Tudo da lista "nunca usar" do `MOODBOARD §8`: cara de celebridade, sorriso de catálogo, CGI/uncanny, olhos de anime, pele plástica, overgrading.

## Como usar
- Soltar as imagens aqui com nome descritivo (ex.: `ref_jawline_natural_01.jpg`, `ref_skin_pores_02.jpg`).
- Estas imagens são **referência de comparação**, não candidatos. Não recebem nota.
- Para ferramentas image-to-3D (ex.: CC5), uma referência pode virar entrada — registrar isso no `meta.json → entrada.imagem_referencia` do candidato.

> **Honestidade:** nenhuma imagem foi gerada/criada por este sprint. Pasta inicia vazia, aguardando garimpo.
