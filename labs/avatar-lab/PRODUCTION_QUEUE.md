# PRODUCTION QUEUE — Kai

Fila **oficial** de produção dos candidatos ao Kai. Esta é a **única** lista de status de produção do lab. Aqui só vive **status** — nada de notas, nada de vencedor.

- **Notas / scores:** `candidates/MASTER_RANKING.md` (avatar inteiro) · `face-lab/FACE_RANKING.md` (rosto).
- **Como produzir cada candidato:** `GENERATION_WORKFLOW.md` (fluxo) + `PRODUCTION_CHECKLIST.md` (passo a passo).
- **Slots = alvos de produção.** A pasta do candidato é criada (copiando `_TEMPLATE/`) **quando a geração começa** — não antes. Linha na fila ≠ asset existente.

|:--:|---|
